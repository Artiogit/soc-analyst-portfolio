# SOC Analyst Portfolio

This repository showcases hands-on cybersecurity projects focused on
Security Operations Center (SOC) activities.

## Skills Demonstrated
- Log analysis (Windows, Firewall)
- Phishing email investigation
- SIEM alert triage (Splunk)
- Incident response workflows
- MITRE ATT&CK mapping
- Threat intelligence analysis

## Tools Used
- Splunk
- Microsoft Defender
- VirusTotal
- Any.Run
- WHOIS / DNS tools
- TryHackMe labs (documented independently)

## Certifications
- ISC2 Certified in Cybersecurity (CC)
- Microsoft SC-900

## Objective
Prepare for a SOC Tier 1 / Junior SOC Analyst role by simulating
real-world security incidents and documenting analysis and response.
